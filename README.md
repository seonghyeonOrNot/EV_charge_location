# EV_charge_location
### 데이터 분석 사용 툴 및 라이브러리
colab. pandas, numpy, folium

##### folium이란?
- 인터렉티브 맴 생성하는 라이브러리
- 위치 데이터 시각화 가능
- 다양한 지도 스타일 제공, 웹 브라우저에서 인터랙티브한 기능 지원
- 지도 위 마커, 원, 다각형, 폴리라인 등 다양한 그래픽 요소 추가 가능
- 팝업, 툴팁, 클릭 이벤트 등 사용가능
